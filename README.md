# Physics-Informed Neural Operators for Option Pricing
- A preliminary study
- Author: Zefang Li and Zhan Liu,
- Contact: zli299(at)jh(dot)edu

In detail, we present a Fourier Neural Operator (FNO) for pricing of European call options. For comparison, a deep neural operator, a concolutional neural operator, and a vanilla neural operator are used as benchmarks.
